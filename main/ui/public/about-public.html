About Our Capstone Project

CS 461 Senior Capstone – Oregon State University

Project Description

This project is part of the CS 461 Senior Capstone at Oregon State University. Version 2 builds upon the original Interactive Survey Tool for Data Visualization, continuing its development as a web-based survey platform that enables researchers to create interactive, data-driven questionnaires with dynamic visualizations for richer and more precise participant insights.

Project Overview

The goal of this project is to develop a custom web-based survey tool that allows researchers at Oregon State University (and other organizations) to create interactive surveys featuring advanced data visualizations. Unlike existing platforms such as Qualtrics, this tool supports dynamic elements including zooming, panning, and selecting regions on visual assets such as SVGs.

The platform emphasizes usability, accessibility, and data security, aiming to enhance both the researcher and participant experience while addressing the limitations of current survey tools.
